# @rnboost/docs

The documentation website for RNBoost CLI. Built with Vite + React + React Router.

## Develop

```bash
npm install
npm run dev --workspace @rnboost/docs
```

Open http://localhost:5173.

## Build

```bash
npm run build --workspace @rnboost/docs
```

The static site is emitted to `packages/docs/dist`.
